# PixelPersona

Генератор аватаров и ASCII-арта

Генератор аватаров и ASCII-арта

## Описание

PixelPersona - это веб-приложение для генерации уникальных аватаров и преобразования изображений в ASCII-арт.

## Функциональность

- Генерация аватаров по имени/нику
- Конвертация изображений в ASCII-арт
- Поддержка различных стилей и тем
- Экспорт в различные форматы

## Сборка и запуск

### Требования

- Java 11 или выше
- sbt (Simple Build Tool)
- Node.js (для запуска разработческого сервера)

### Сборка проекта

```bash
./build.sh
```

Этот скрипт выполнит следующие шаги:
1. Очистку предыдущей сборки
2. Компиляцию проекта
3. Запуск тестов
4. Создание JavaScript бандла

### Запуск в режиме разработки

```bash
./dev-server.sh
```

Этот скрипт запустит разработческий сервер на порту 8080 с отслеживанием изменений в коде.

### Структура проекта

```
pixel-persona/
├── build.sbt              # Конфигурация сборки
├── project/               # Файлы проекта sbt
├── src/                   # Исходный код
│   ├── shared/            # Общий код для JVM и JS
│   ├── jvm/               # Код только для JVM
│   └── js/                # Код только для JavaScript
└── target/                # Сгенерированные файлы
```
